# Happy_RC_Driver
**スマホでラジコンカーが操作できる**アプリ（送信機）とレシーバ（受信機）を作っています。

ラジコン用のプロポ（送信機）はポケットに入れて持ち運ぶことできませんが、スマホならポケットに入れて持ち運ぶことができます。そのためラジコンカー本体だけ持ち運べば、いつでもどこでも手軽にラジコンを始めることができます！さらに、スマホアプリと受信機は自分の**好みに合わせてカスタマイズ**可能です！
